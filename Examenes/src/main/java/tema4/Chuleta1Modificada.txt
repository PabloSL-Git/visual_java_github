Mostrar mensajes y pedir datos (JOptionPane)

Pedir String:

Pedir entero:

Pedir float:
float n = Float.parseFloat(JOptionPane.showInputDialog("Introduce un flotante:"));

Pedir long:
long n = Long.parseLong(JOptionPane.showInputDialog("Introduce un número largo:"));

Pedir short:
short n = Short.parseShort(JOptionPane.showInputDialog("Introduce un número corto:"));

Pedir byte:
byte n = Byte.parseByte(JOptionPane.showInputDialog("Introduce un número byte:"));

Pedir boolean:
boolean b = Boolean.parseBoolean(JOptionPane.showInputDialog("true o false:"));

Pedir char:
char c = JOptionPane.showInputDialog("Introduce un carácter:").charAt(0);

Mostrar mensaje:
JOptionPane.showMessageDialog(null, "Texto");

printf — Formatos

%d → Entero
%f → Decimal
%s → String
%c → Carácter
%b → Booleano
%n → Salto de línea
%% → Símbolo %

Ejemplos:
System.out.printf("Nombre: %s\n", nombre);
System.out.printf("Precio: %.2f €\n", precio);

Convertir número a String:

Convertir char a String:

Convertir boolean a String:

Convertir cualquier tipo a String (método universal):

Convertir int a String con formato:

Convertir double con 2 decimales:
String s = String.format("%.2f", numero);


public static void moverCeros(int[] arr) {
    int index = 0;

    // Copiar los números que NO son cero al principio
    for (int num : arr) {
        if (num != 0) {
            arr[index++] = num;
        }
    }

    // Rellenar el resto con ceros
    while (index < arr.length) {
        arr[index++] = 0;
    }
}
